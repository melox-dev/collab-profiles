# 🧑‍💻 collab-profiles

Site colaborativo de perfis pessoais feito em HTML e CSS.

## 👀 Visualizar o site

🔗 [phc-silva.github.io/collab-profiles](https://phc-silva.github.io/collab-profiles)

## 📋 Sobre o projeto

Site com cards de perfil personalizados para cada integrante, contendo foto, bio, links para redes sociais e música favorita no Spotify.

## ✨ Funcionalidades

- Cards de perfil personalizados
- Links para Instagram e TikTok
- Player do Spotify integrado
- Design escuro e responsivo

## 🛠️ Tecnologias

- HTML5
- CSS3
- Spotify Embed API
